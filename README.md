# 최수호 AI Service Planning Portfolio

카카오뱅크 AI 서비스 기획 어시스턴트 지원용 포트폴리오 페이지입니다.
프롬뱅크 프로젝트를 중심으로 문제 정의, 기능 기획, MVP 검증, 시장성, 리스크와 로드맵을 정리했습니다.

## 구성

- `index.html`: 포트폴리오 본문
- `style.css`: 레이아웃, 반응형, 모션 스타일
- `script.js`: 인트로 전환, 스크롤 애니메이션
