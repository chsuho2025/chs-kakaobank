# 프롬뱅크 Portfolio

카카오뱅크 AI 서비스 기획 어시스턴트 지원용 포트폴리오 페이지입니다.
프롬뱅크 PDF 포트폴리오를 웹에서 책처럼 넘겨볼 수 있도록 구성했습니다.

## 구성

- `index.html`: 포트폴리오 본문
- `style.css`: 레이아웃, 반응형, 모션 스타일
- `script.js`: 인트로 전환, 슬라이드 넘김 기능
- `assets/slides`: 목차 페이지를 제외한 PDF 슬라이드 이미지
- `assets/fonts`: Pretendard 웹폰트 파일

## 배포 방법

이 폴더의 파일을 GitHub 저장소에 올린 뒤 Vercel에서 정적 사이트로 배포하면 됩니다.
